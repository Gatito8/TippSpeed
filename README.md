**TippSpeed** (from German: *Tippgeschwindigkeitstest*) is a Java Swing application that measures your typing speed and highlights correct and incorrect characters in real time.  
The app was built in **Java 21** using **NetBeans** with an **MVC architecture**.

---

## ‚öôÔ∏è Features
- Real-time countdown timer (using `javax.swing.Timer`)
- Character-by-character color highlighting (green = correct, red = wrong)
- Customizable text generation (uppercase, lowercase, numbers, marks, traps)
- Adjustable difficulty levels and typing duration
- Clean, responsive Swing UI
